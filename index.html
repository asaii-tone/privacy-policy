サポート時

■日本語

■English

In the support email, the terminal type, OS version, etc. for problem solving will be described as the text.
No personally identifiable information will be sent.

アプリの利便性向上のため、匿名で特定できない範囲で最新の注意を払い、アクセス解析をしております。
例)アプリのクラッシュ時にどのような原因でクラッシュしたかを匿名で送信して、バグの修正に役立たせております。

■English

In order to improve the convenience of the application, we pay the latest attention to the extent that it cannot be identified anonymously and analyze the access.
ex) When the application crashes, we anonymously send the cause of the crash to help fix the bug.
